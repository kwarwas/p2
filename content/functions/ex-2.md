# ASCII

Napisać implementację trzech funkcji 

    char toLower(char x);
    char toUpper(char x);
    char toggle(char x);

które będą działać następująco:

    toLower('R') == 'r'
    toLower('r') == 'r'
    toUpper('r') == 'R'
    toUpper('R') == 'R'
    toggle('R') == 'r'
    toggle('r') == 'R'
